# Live2DPet Global Quick Start

Default language: Chinese (`README.md`).

Official docs:
- Chinese (default): [README.md](README.md)
- English: [README.en.md](README.en.md)
- Japanese: [README.ja.md](README.ja.md)

For other languages, open this page in your browser and use built-in translation.

---

## 1) Chinese / 中文（默认）

1. 安装 `Node.js 18+`（建议 20 LTS）。
2. 下载发布版：从 Releases 下载 `Live2DPet.exe`，双击运行。
3. 源码运行：
   - `npm install`
   - `node launch.js`
4. 在设置页填 API（`base_url` / `api_key` / `model`）。
5. 选择模型后点击“启动宠物”。
